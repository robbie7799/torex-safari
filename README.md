# 🦁 TOREX SAFARI - Modern Travel & Adventure Theme

**A comprehensive safari-inspired travel theme with Uganda Tourism Board (UTB) and Uganda Registration Bureau (URBS) integration.**

## 🌍 Overview

TOREX SAFARI is a modern, vibrant theme designed for travel and tourism websites, featuring:

- **Safari Wildlife Aesthetics** - Lions, elephants, giraffes, and exotic creatures
- **Uganda Tourism Branding** - UTB official colors and identity
- **Monster/Character Theme** - Fun adventure mascots
- **Responsive Design** - Mobile-first approach
- **Adventure Booking** - Tours, safaris, and travel packages
- **Cultural Integration** - URBS compliance and Ugandan heritage

## 📦 Installation

```bash

git clone https://github.com/robbie7799/torex-safari.git

# Navigate to directory
cd torex-safari

# Open in browser
open index.html
```

## 🎯 Quick Start

### Include Theme CSS
```html
<link rel="stylesheet" href="css/variables.css">
<link rel="stylesheet" href="css/theme.css">
<link rel="stylesheet" href="css/components.css">
```

### Include Theme JavaScript
```html
<script src="js/theme.js"></script>
<script src="js/bookings.js"></script>
```
